Target port
Available properties and export format depend on the target port. Configure the target in Map Metadata. The export format is Binary WAD or UDMF/TEXTMAP.

Limitations:
BEHAVIOR, SCRIPTS, DIALOGUE lumps, and Custom properties are NOT preserved. Floating-point coordinates are quantized to integers.

Supported ports:
DOOM / BOOM / HEXEN / ZDOOM / GZDOOM

Legacy ports:
Exported maps need to be built in a map editor with a node builder (e.g. Ultimate Doom Builder) before they can be played in legacy ports.

Void sectors:
The editor automatically creates all enclosed sectors. To exclude a sector, enable Is Void in the inspector.

Users
• PM: Send a private message.
• Limit: Restrict editing to the selected area.
• Kick: Disconnect the user.
• Ban: Block the user from the server.

Welcome to the editor.
Navigation
  • W / A / S / D → Move FPS-style
  • Shift + A / D → Strafe
  • Right Mouse (2D) → Pan view
  • Right Mouse (3D) → Flight
  • Mouse Wheel (2D) → Zoom in / out
  • Tab → Swap 2D / 3D viewport
Selection
  • Left Mouse → Select
  • Shift + Left Mouse → Toggle select
  • Ctrl + Left Mouse → Brush select
  • Left Drag (on selection) → Move selection
  • Left Drag (right) → Box select (contains)
  • Left Drag (left) → Box select (intersects)
  • Middle Mouse → Brush deselect
  • 1 → Grow selection (connected)
  • 2 → Shrink selection (connected)
  • 3 (2D / 3D) → Select connected (hovered)
Snapping
  • 4 / 5 → Snap grid scale down / up
  • Shift (hold) → Enable polar snap
  • Ctrl (hold) → Disable grid snap
Mode switching
  • Space / Esc → Select mode / Deselect
  • V → Line mode
  • B → Rectangle mode
  • C → Ellipse mode
  • E → Extrude mode
  • Q → Thing mode
Transform modes
  • Left Mouse → Confirm transform
  • G → Confirm & continue moving
  • R → Confirm & continue rotating
  • H → Confirm & continue scaling
  • Right Mouse → Move transform pivot
Drawing modes
  • Left Mouse → Confirm drawing
  • Middle Mouse → Add & continue
Geometry editing
  • N → Flip selection void
  • F → Equalize selection height
  • Ctrl + F → Flip hovered side
  • Scroll (3D) → Change sector height
  • Shift + Scroll (3D) → Finetune height
Texture editing
  • Middle Drag → Scroll texture
  • T → Auto-align horizontally
  • Y → Auto-align vertically
  • Auto-align is relative to first selected
  • Shift + C → Copy texture
  • Shift + V → Paste texture
  • Texture targets are selected in 3D
Deletion
  • X → Dissolve or delete hovered
  • Delete → Delete selected
  • Shift + Delete → Dissolve selected
Clipboard
  • Copy and paste work across tabs
  • Ctrl + C → Copy
  • Ctrl + X → Cut
  • Ctrl + V → Paste
History
  • Ctrl + Z → Undo
  • Ctrl + Y → Redo
Tabs
  • F1 - F7 → Switch tab
